# generate-qrcode-node
Node &amp; express api to generate QR Code from user data

<b>Create user and generate their qr code</b><br>
/api/users <br>
<b>Retrieve user qr code</b> <br>
/api/users/{user_name}/qrc

npm install <br>

  qrcode

  fs
